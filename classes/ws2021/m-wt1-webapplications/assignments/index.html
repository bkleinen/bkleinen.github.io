2. CI/CD

Set up Continuous Integration and - Deployment for your project. You don’t need to have 100% or even 60% code coverage necessarily, but consider carefully what to do and answer the questions in your report.

  • Set up Continuous Integration with
    • Automated Tests: Which tests should you have to most effectively ensure that your app is still working after a change? Which are the most important features, how can you test most effectively? (think UI Tests vs Unit test, see The Practical Test Pyramid)
    • Code Quality Assurance - find and test one code quality assurance tool
    • How does your deployment pipeline look like? which steps are automatic, which require manual interaction and if, which one? see DeploymentPipeline.
  • Set up Continuous Deployment to  HTW Server using Docker Containers.
